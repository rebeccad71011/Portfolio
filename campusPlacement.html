Campus Placement Predictor

A web app built to help college faculty predict student placements

View Demo
View on GitHub



Project Overview

A machine learning based web application that allows students to predict the status of their placement based on their respective profiles. The training and placement officer (TPO) can view the prediction results in an interactive dashboard, and also filter and search for students based on various properties. An admin can easily manipulate and verify the data available in the application, if need be.
